HVAC Unit Converter

Welcome to Global HVAC Solutions LLC — your quick engineering unit converter (CMH↔CFM, kW↔BTU/h, °C↔°F, Pa↔in. wg, and more).

📥 Download the Python Script

⬇️ Download converter.py (right-click → Save Link As)

🚀 Quick Usage Example

python converter.py kw_to_btuh 5.5
# Output: 5.5 kW = 18,766.78 BTU/h
